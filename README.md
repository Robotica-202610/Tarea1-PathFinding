## Instrucciones de ejecución

> Nota: Correr `build.py` creará un archivo ejecutable para su dispositivo en el directorio `dist`.
> La ejecución del archivo ejecutable no requiere de la instalación de Python ni de paquetes adicionales, pero será más lenta que correr el código directamente con Python.
> EL archivo `build.py` fue realizado con la ayuda del modelo de IAG Claude Opus 4.5.

- Windows:
  ```powershell
  python .\build.py
    ```
- MacOS/Linux:
- ```zsh
  python3 ./build.py
    ```

1. Instalar los paquetes de python necesarios para la ejecución

   - Windows:
      ```powershell
      pip install numpy pandas 
       ```
   - MacOS/Linux
      ```zsh
      pip3 install numpy pandas 
       ```

2. Correr el archivo `__main__.py` ubicado en la carpeta `src` para iniciar la simulación.

   - Windows:
      ```powershell
      python src\__main__.py
       ```
   - MacOS/Linux
      ```zsh
      python3 src/__main__.py
       ```

3. Escoger entre el CLI o el GUI para interactuar con la simulación.
